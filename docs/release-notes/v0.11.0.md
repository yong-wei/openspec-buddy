# v0.11.0

## Changes

- Added `verify-current-head-review-request.sh` to require a fresh
  current-head PR review request before review waiting begins.
- Extracted review-request freshness detection into `review-request-state.mjs`
  so `request-pr-review.sh` and `wait-for-review-clear.sh` use the same
  current-head standard.
- Hardened `wait-for-review-clear.sh` so it fails before sleeping when a
  review-fix commit has resolved old threads but has not requested a new review
  for the current head.
- Added `request-pr-review.sh --context-file` so review-fix follow-up requests
  append structured context after the fixed configured review request string.
- Updated Buddy Auto review-fix guidance to require:
  `review-response-gate.sh`, then `request-pr-review.sh`, then
  `wait-for-review-clear.sh`.
