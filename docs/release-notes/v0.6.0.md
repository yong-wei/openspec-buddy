# v0.6.0

## Highlights

- Added a foreground review-wait helper so Buddy Auto can wait for current-head
  review clearance without main-thread polling or fixed idle rounds.

## Changes

- Added `wait-for-review-clear.sh`, which performs one foreground blocking wait,
  uses low-frequency REST checks, and invokes `verify-review-clear.sh` only on
  first check, review-state changes, or timeout.
- Added review wait configuration defaults for initial wait, poll interval, and
  maximum wait duration.
- Updated Buddy Auto review guidance to use the wait helper and to keep the
  main agent silent while the helper runs.
- Normalized configured reviewer matching so `chatgpt-codex-connector` and
  `chatgpt-codex-connector[bot]` are treated as the same Codex reviewer.
- Added eval coverage for the wait helper and Codex bot reviewer matching.
