# v0.1.0

## Highlights

- Moved `openspec-buddy` and `openspec-buddy-auto` into a versioned public
  repository.
- Added npm package metadata and a CLI for installing the skills into common
  agent skill roots.
- Added first-project configuration generation through `openspec-buddy init`.

## Changes

- Added `copy` and `symlink` install modes.
- Added install targets for `$HOME/.agents/skills`, Codex skills, and
  project-local `.agents/skills`.
- Replaced machine-local skill path examples in skill documentation with a
  portable skill-directory placeholder.

## Verification

- `npm test`
- `npm pack --dry-run`
